# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-20

### Added
- Initial release of sb-theme-switcher
- Automatic UI adaptation: toggle for 2 themes, dropdown for 3+ themes
- Support for Storybook 7.x, 8.x, 9.x, and 10.x
- Theme persistence via localStorage
- Cross-tab synchronization
- Docs mode support with DocsContainer wrapper
- TypeScript support with full type definitions
- Customizable themes, icons, and colors
- `useTheme()` hook for reading current theme in components
- `withTheme` decorator for preview
